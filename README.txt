This program is used to assist one of the professors to grade the final exam of his students. The exam consists of 20 multiple-choice questions. Each question has one of four possible answer which is : A, B, C, D. We are also required to prepare a data file named "StudentAnswer.dat" and a text file named "CorrectAnswer.txt". The program will read the students' answers and the correct answers from the files and print out the result onto the screen and the file output.
